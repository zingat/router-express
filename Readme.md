```js
var express = require('express')
var routerExpress = require('router-express')

var app = express()

var routes = [
  {
    name: 'homepage',
    method: 'get',
    url: '/',
    module: 'static/home'
  }
]

global.Router = new routerExpress(routes)
Router.bind(app)
```

### Install
```sh
npm install router-express
```

### Features

* Name-based routing
* Reverse routing
* Parameter injection and default parameters
* HTTP methods support
